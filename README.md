# To-Do List App

## Description:
This is a simple to do list app that allows the user to add, edit and delete tasks. The task can be marked as complete. 

## Programming Technologies:
<li>Python</li>
<li>Django</li>
<li>HTML</li>
<li>CSS</li>
<li>Bootstrap</li>
